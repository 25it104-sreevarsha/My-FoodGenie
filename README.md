# 🍽️ OrderIt – AI Powered Food Delivery Website

OrderIt is a full-stack AI-powered Food Delivery Website built using the MERN Stack. It enables users to browse restaurants, explore menus, place food orders, make secure Stripe payments, and experience AI-powered features like food description generation and restaurant review summaries.

## 🚀 Features

- 🔐 User Authentication (Login & Register)
- 🍴 Restaurant & Menu Browsing
- 🛒 Shopping Cart
- 💳 Secure Stripe Payments
- 📦 Order Management
- 🤖 AI Food Description Generator
- 🧠 AI Review Summary (Groq AI)
- ☁️ Cloudinary Image Uploads
- 👨‍💼 Admin Dashboard Features
- 📱 Fully Responsive Design

## 🛠️ Tech Stack

### Frontend
- React.js
- Redux Toolkit
- Bootstrap
- Axios
- Vite

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

### AI & Third-Party Services
- Groq AI
- Stripe
- Cloudinary

## ⚙️ Installation

### Clone Repository

```bash
git clone 
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file inside the backend folder and configure the required API keys.
